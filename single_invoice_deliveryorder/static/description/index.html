Single Invoice From Multiple Delivery Order

The primary purpose of this module is to easily create single Invoice from the Multiple Delivery Orders. You don't need to create Invoice manually.Just select Multiple Delivery of same contacts and click on action for "Create Single Invoice" and you are ready to go.

Below listed are features of “Single Invoice From Multiple Delivery Orders ” module:

Prerequisites before Installation

There are no prerequisites required before the Installation

Configurations

As shown in the image when user select two/more orders from the Delivery Order and click on the “Action” tab they will be able to create single Invoice by clicking on "Create Single Invoice”.

When you click on the “Create Single Invoice” you will be asked to create single Invoice from multiple Multiple Delivery.

After clicking on "Yes" option,following Invoice of multiple selected Multiple Delivery will be created.
